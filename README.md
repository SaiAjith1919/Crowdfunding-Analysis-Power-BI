# 📊 Crowdfunding Campaign Analysis Dashboard (Power BI)

## 🚀 Project Overview

This project presents an **interactive Power BI dashboard** designed to analyze crowdfunding campaign data. It provides insights into **campaign performance, funding success rates, category trends, and backer behavior**, enabling data-driven decision-making for campaign creators and investors.

---

## 🎯 Objectives

* Analyze **successful vs failed campaigns**
* Track **funding trends over time**
* Identify **top-performing categories**
* Understand **backer engagement and contribution patterns**
* Evaluate factors influencing campaign success

---

## 🛠️ Tools & Technologies Used

* Power BI Desktop
* Power Query (ETL & Data Cleaning)
* DAX (Data Analysis Expressions)
* Data Modeling (Star Schema)
* GitHub + Git LFS

---

## ⚙️ Techniques Implemented

### 🔹 Data Cleaning & Transformation (Power Query)

* Removed duplicate and null records
* Standardized date formats (launch & deadline)
* Created calculated columns such as:

  * Campaign Duration
  * Funding Gap
* Filtered invalid or incomplete campaigns

---

### 🔹 Data Modeling

* Built a structured model with:

  * Fact Table → Campaign Data
  * Dimension Tables → Category, Country, Time
* Established relationships for efficient analysis

---

### 🔹 DAX Measures (Advanced Analytics)

Developed key metrics using DAX:

* Total Funding Amount
* Total Backers
* Success Rate (%)
* Average Funding per Campaign
* Funding Goal vs Achieved Amount
* Campaign Duration Analysis
* Year-wise Growth Trends

---

## 📊 Key KPIs

* 💰 Total Funds Raised
* 🎯 Funding Goal Achievement %
* 📈 Success Rate
* 👥 Total Backers
* 📊 Average Contribution per Backer

## 📊 Dashboard Visualizations

### 🔹 Campaign Performance

* KPI cards displaying funding, success rate, and backers
* Donut/Pie chart showing **successful vs failed campaigns**

---

### 🔹 Category Analysis

* Bar chart of **top categories by funding**
* Category-wise success rate comparison

---

### 🔹 Time-Based Trends

* Line chart showing **funding trends over time**
* Monthly/Yearly campaign performance

---

### 🔹 Geographic Analysis

* Map visualization showing **funding by country**
* Regional comparison of campaign success

---

### 🔹 Backer Insights

* Distribution of backers across campaigns
* Average contribution per backer

---

## 📈 Key Insights Derived

* Identified categories with **highest success rates**
